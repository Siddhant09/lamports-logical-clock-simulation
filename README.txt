------------------------------------------------------
INSTRUCTIONS FOR COMPILING AND RUNNING THE APPLICATION
------------------------------------------------------

A make file has been created with the following commands:

make: To compile and create .class files

make clean: To delete old .class files

make server: To run the server

======================================================

--------------------------------------
INSTRUCTIONS FOR EXECUTING THE PROGRAM
--------------------------------------

1. The application asks the user to enter the desired number of random events to be generated among the four process objects.
2. The events are generated randomly once a number is entered.
3. The list of events generated will be displayed in the terminal.
4. The time stamps of all the process objects will be displayed in the terminal.
5. The application terminates.
